# Gestor_biblioteca_Personal


## ¿Que es?

Aplicación de consola desarrollada en C++ para administrar un registro de libros utilizando std::list como estructura de datos principal. El sistema permite realizar operaciones CRUD completas (agregar, consultar, modificar y eliminar libros), además de generar un archivo de texto con todos los registros. Incluye validación robusta de entrada, normalización de texto a mayúsculas y un menú interactivo diseñado para facilitar la navegación del usuario.

## Características

* Registro de libros con título, autor, género, editorial y número de páginas.
* Conversión automática de texto a mayúsculas para búsquedas consistentes.
* Visualización del catálogo en formato tabular.
* Búsqueda exacta por título.
* Modificación de cualquier campo del libro
* Eliminación segura mediante iteradores.
* Validación de entrada numérica para evitar errores.
* Generación de archivo .txt con todos los libros registrados.
* Menú interactivo con control de errores y pausas seguras.

### Tecnologías y conceptos aplicados
* C++ 17
* std::list
* Manejo de archivos con fstream
* Validación de entrada (cin, cin.clear(), cin.ignore() )
* Programación modular
* Diseño de menús y control de flujo

## Mi Objetivo
Demostrar dominio de estructuras de datos dinámicas, manejo de archivos, validación de entrada y diseño de aplicaciones de consola. Este proyecto forma parte de un portafolio orientado a mostrar habilidades prácticas en C++ y buenas prácticas de programación.
